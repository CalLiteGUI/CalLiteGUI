3.5. Regulations Dashboard

The Regulations dashboard allows the user to select different regulations that will be active in the current scenario.  There are three sub tabs here for activating different regulations, for D-1641 regulations, Biological Opinion Reasonable and Prudent Alternative (RPAs), and Other regulations.

With the exception of the Clear Creek RPA, all of these regulations are for flows or water quality in or near the Delta.  Appendix B of the User's Reference Manual has a detailed description of all of the regulations on this dashboard.
