3.5. Regulations Dashboard

The Regulations dashboard allows the user to select different regulations that will be active in the current scenario.  There are three sub tabs here for activating different regulations, for D-1641 regulations, Biological Opinion Reasonable and Prudent Alternative (RPAs), and Other regulations.

WARNING: Users should be careful when setting user-defined standards (on the D-1641 or Other regulation tab). Please be cautious that user specified values may cause non-feasible operations although the model may keep running. Please check with modeling/operations experts to ensure that operations remain reasonable for these inputs.

With the exception of the Clear Creek RPA, all of these regulations are for flows or water quality in or near the Delta.  Appendix B of the CalLite Reference Manual has a detailed description of all of the regulations on this dashboard.
