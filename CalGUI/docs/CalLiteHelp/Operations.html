3.6. Operations tab

The operations tab has options relating to CVP and SWP system operations, as follows:

-CVP Operations and Facilities.  The checkboxes here can be used to turn on two different wheeling operations (CVC Wheeling and Joint Point of Diversion) and also the Delta-Mendota Canal(DMC)/California Aqueduct Intertie.  For more description of wheeling operations, see section ... of the User's Reference Manual.  The Intertie project allows the DMC and California Aqueduct to share conveyance capacity, thus increasing export capacity.

-CVP/SWP Allocation Methods.  Here the user can choose between different CVP and SWP allocation methods.  Currently the only available allocation method is Water Supply Index/Delivery Index (WSI/DI) Curves, but other methods will be available in future versions of CalLite. Default WSI/DI curves for both CVP and SWP can altered here by clicking on the appropriate box and editing the values in table on the right side of the screen.  For more information about CalLite allocation methods, see Appendix E of the User's Reference Guide.
