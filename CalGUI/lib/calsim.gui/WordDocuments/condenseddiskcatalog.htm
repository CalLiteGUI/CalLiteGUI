Condensed Disk Catalog

A condensed catalog is a list of pathnames where the dates of time series data are grouped together and show a date span instead of a date D part for each record.  To create a condensed catalog:     

  1. After opening a HEC-DSS file, select the Advanced → Condensed Disk Catalog → New menu option.
  2. A message will appear stating the catalog has been updated and the number of records it contained, see below.


    Catalog file update window

To view a Condensed Disk Catalog:

  1. After opening a HEC-DSS file, select the Advanced → Condensed Disk Catalog → View menu option.  If the catalog does not exist, it will be automatically created.
  2. A text dialog will open displaying the Condensed Disk Catalog, see below. 
  3. You can print the catalog through the text dialog by selecting its File →  Print menu option. 
  4. The dialog contains File menu commands to Insert .txt, Save As .txt, change the Font, and Print Setup.  It also contains a Find menu that can facilitate a search of the catalog displayed.


    Condensed catalog text output

To print a Condensed Disk Catalog:

  1. After opening a HEC-DSS file, select the Advanced → Condensed Disk Catalog → Print menu option.
  2. A Print properties dialog will open (see below); click OK to send the catalog to the printer.


    Print properties dialog
