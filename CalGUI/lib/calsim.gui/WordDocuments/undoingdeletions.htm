Undoing Deletions

Deleted records may be recovered as long as the HEC-DSS file has not been squeezed.  When a file is squeezed, all deleted records are physically removed.

You can undelete records in three ways: undelete all records in the file, select the records to undelete from a list, or undelete the records that you just deleted.

To undelete records from a HEC-DSS file:

  1. From the Edit menu, select Undelete → All, Undelete → Select, or Undelete → Last Deleted. 
  2. To undelete all records selected All.  A dialog box will appear to confirm that you want to undelete all of the records in the file.  Click OK.  A confirmation message will appear that states the records have been undeleted.
  3. To undelete the records that you have just deleted, select Last Deleted.  A dialog box will confirm that you want to undelete the records that you just deleted.  Click OK.  A confirmation message will appear that states the records have been undeleted.
  4. If you choose Select, a list of the previously deleted pathnames will open, as shown below.


    HEC-DSSVue Undelete Records Selection Dialog Box
        
  5. Check the boxes next to the records that you want to undelete, or press the Select All button to undelete all records in the list.  You can unselect the checked records by pressing the Unselect All button.  Once you have made your selection, press the OK button to undelete those records and close the dialog.  Alternatively, you can press Apply to undelete the records and leave the dialog open.  A confirmation message will appear stating that the records have been undeleted.
