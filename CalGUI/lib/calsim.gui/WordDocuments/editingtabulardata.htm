Displaying and Editing Data


Editing Tabular Data

In HEC-DSSVue, you can edit data directly in tables, but first you must set the table to allow editing. 

From the Edit menu of the table, select Allow Editing (see below).  This will set the table editable and allow you to manually edit the listed data.  When this option is checked, the background of non-editable table cells display grey and the Cut, Paste, Insert Rows, and Delete Rows commands become available in the Edit menu.
 

Edit menu - Allow Editing option

If the Allow Editing option is disabled (grayed-out), you may not have permission to write to the file or the table was started from a dialog that does not have access to write to the file (such as a plot window).

Changes made to a table can be saved by selecting the File → Save or File →  Save As menu option.  If you do not save your data before you close the table window, HEC-DSSVue will prompt you to save your changes before it closes the window.
