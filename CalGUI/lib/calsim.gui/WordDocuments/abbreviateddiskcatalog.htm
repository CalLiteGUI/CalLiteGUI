Abbreviated Disk Catalog

An abbreviated catalog is a sorted list of record pathnames in a HEC-DSS file.  To create an Abbreviated Catalog:

  1. After opening a HEC-DSS file, select the Advanced → Abbreviated Disk Catalog → New menu option.
  2. A message will appear stating the catalog has been updated and the number of records it contained, see below.


    Catalog file update window

To view an Abbreviated Disk Catalog:

  1. After opening a HEC-DSS file, select the Advanced → Abbreviated Disk Catalog → View menu option.
  2. A text dialog will open displaying the Abbreviated Disk Catalog, see the figure below. 
  3. You can print the catalog through the text dialog by selecting its File →  Print menu option. 
  4. The dialog contains File menu commands to Insert .txt, Save As .txt, change the Font, and Print Setup.  It also contains a Find menu that can facilitate a search of the catalog displayed.


    Abbreviated catalog text output

To print an Abbreviated Disk Catalog: 

  1. After opening a HEC-DSS file, select the Advanced → Abbreviated Disk Catalog → Print menu option.
  2. A Print properties dialog will open, see below; click OK to send the catalog to the printer.
