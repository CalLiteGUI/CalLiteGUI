Scripting > Executing Scripts


Script Menu

The Script Selector (see below) can display a list of available scripts and execute a script by selecting it from the menu.  The Script Menu, shown in the second figure below, is not made visible on the main HEC-DSSVue menu bar until scripts are set to be displayed on the menu bar from the Script Editor, available from the Tools menu.   To have a script displayed in the Script Menu, simply edit the script in the Script Editor, select the Show in Scripts Menu checkbox at the top of the editor (see the  third the figure below)
