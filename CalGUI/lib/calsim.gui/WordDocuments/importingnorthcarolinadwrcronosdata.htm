Importing North Carolina DWR CRONOS Data

The North Carolina Department of Water Resources developed a plug-in to import data from the CRONOS database into HEC-DSS.  This plug-in is supported by North Carolina DWR.  For questions, please contact them directly.  The web site is http://www.ncwater.org/wrisars/dss.

The CRONOS Plug-in, developed by NC DWR, can search and retrieve the water and weather data for North Carolina and the surrounding states from NC Climate Retrieval and Observations Network of the Southeast Database (CRONOS) and store the data into a HEC-DSS file. The map and table search engine in Water Resources Information, Storage, Analysis, and Retrieval System (WRISARS) can be used with the CRONOS Plug-in to enhance the search capability. To import data from CRONOS:

  1. In HEC-DSSVue, create a new dss file or open an existing one.
  2. In the HEC-DSSVue main window, click the Data Entry menu, then NC DWR CRONOS from the Import popup, as shown below.


    Importing ND DWR CRONOS
      
  3. In the newly opened NC DWR CRONOS Download panel, select a Data Type, as shown below.


    Data Type Selection
       
  4. Select a Data Table, see below.


    Data Table Selection
      
  5. Click the  button and then select a state from the drop-down list, shown below, or click the  button and then select a HUC. Click OK to search the stations and you will see the selected stations in the station table.


    Select State
     
  6. If you check check box, then the station table will be cleared when you select another group of stations.
  7. If you want to do a more specific search, click the  button. It will lead you to the WRISARS search page.
  8. You can save the station table to a .cronos text file by clicking the File menu, then Save Station Table.
  9. You can open a previously saved station table from an existing .cronos text file by clicking the File menu and then Open Station Table.
  10. You can open a station table directly by clicking a file name (e.g. StationTableC.cronos) below the Close section in the File menu.
  11. Select stations in the station table.
  12. You can click the column title to sort the station table by that column, e.g. click Available End to sort by the available end timestamps. The columns in the table are shown in the figure below.


    Column Header Titles
      
  13. You can also use the  button or the  button to select or unselect all rows in the station table, respectively.
  14. You can delete the checked rows from station table by clicking the Edit menu Edit, then Delete Checked Rows.
  15. You can also add rows in front of the current row by clicking the Edit menu and then Insert Rows.
  16. If you want to use the Station IDs for B parts, click the Edit menu and then click Set B Part to Station ID.
  17. If you want to use the Station Names for B parts, click the Edit menu and then click Set B Part to Station Name. The Stations Names are used for B parts by default.
  18. If you just want to import a part of the data, then set up the Start Date and End Date, otherwise check the Retrieve Period of Record check box.  (You can click button to get the most accurate available dates information.) See the figure below for an example of this toolbar.


    Setting the Time Window for Data Retrieval
      
  19. Click the  button to open the Select Parameters window for the selected stations. The descriptions for the parameters are shown on the right of the window. By default, all parameters are selected. You can click the  button or the   button, or you can select a few parameters manually. If you check the upper checkbox in the All column, then the whole row will be selected automatically. Then click the  button to import the data from the CRONOS database to the DSS file you just opened or created. An example of the Select Parameters window is shown below.


    Select Parameters Window
     
  20. You can click Plot , Tabulate , or Tabulate in MS Excel buttons to open the Select Parameters window, select some parameters, and then import the data and show the data in a plot, table, or Excel directly.
  21. You can also click the View menu, then Tabulate, Plot, or Tabulate with Excel to import the selected station-parameters and show the data in a plot, table or Excel.
  22. Examine the imported or updated data in HEC-DSSVue root window, table, plot, or Excel.
