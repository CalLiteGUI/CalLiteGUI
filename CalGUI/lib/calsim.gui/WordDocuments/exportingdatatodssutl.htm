Exporting Data to DSSUTL

To export data from the DSSUTL format:

  1. Select the pathnames for the data sets that you want to export, then from the Data Entry menu, select Export and then Dssutl Write Data File.
  2. A file selection browser will be displayed.  Navigate to the directory you wish to export the data to, type the name of file and Press Save.  All data will be saved in one file.
