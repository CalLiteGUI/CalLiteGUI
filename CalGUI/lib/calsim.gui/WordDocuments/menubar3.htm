Math Functions > The Math Functions Screen


Menu Bar

Menu options in the Math Functions screen allow you to save or rename data computed with the Math functions, and to display the computed and original data in plots and tables. The Math Functions menus are as follows:
   

File

File menu commands are Save, Save As, and Close. 

Edit

The Edit menu contains the Restore Original Data command.

Display

Use the Display menu to open plots and tables with the Plot and Tabulate commands, and set options for data to display with the Original Data with computed and All Data Sets commands.
