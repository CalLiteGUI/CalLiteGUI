Customizing Plots > Plot Editors and Tools

Configure Plot Editor

The Configure Plot editor (see below) is accessed from the Edit menu of a plot and allows you to customize the data configuration of an individual plot.  You can add and remove axes and add, remove, arrange the order of, and set the weight of viewports in the plot window.

 

Configure Plot Editor

 

When you customize the layout of a plot using the Configure Plot editor, your changes apply only to that individual plot; unless you create a template from the plot’s properties.

 

To access the Configure Plot editor, from the Edit menu, click Configure Plot Layout.
