Import Image and Generic Files

HEC-DSS has a convention for storing image (photos, etc.) and other files in a generic file format.  For this format, the pathname will have a C part that is the name of the file (without directory), a D part that is either “IMAGE” or “FILE”, and an E part that is the extension of the file (e.g., “JPG” or “PDF”). 

 

When you select images or files (generic type) in HEC-DSSVue and select either “plot” or “tabulate”, HEC-DSSVue will launch a photo viewer for image data, or launch the application associated for that file type.  For example, if the file type is “.pdf”, HEC-DSSVue will launch Adobe Acrobat with a copy of that file loaded into it.

 

The easiest way to import both image and generic files is to drag them from Windows Explorer and drop them into the main screen of HEC-DSSVue with an opened HEC-DSS file.  This will display a Save As dialog where you can enter the remaining pathname parts.  You can also import images and generic files from the import function under the Data Entry menu.

 

To import image or generic files into HEC-DSS:

  1. From the Data Entry menu select Import and then either Images or Files (Generic type).
  2. A file selection browser will be displayed, see figure below.  Navigate to the directory and select the files you wish to import.  Select the files and press Open.  You can import multiple files at one time.


    Open Images and Files (Generic type)
     
  3. The Save As Dialog will be displayed.  Enter the Pathname Parts into the A, B, and F boxes.  The complete pathname will automatically appear in the Pathname box.  You can also enter the pathname into the Pathname box; the parts will appear in the Pathname Parts boxes, see below. 


    Save As window
        
  4. To save the records, click OK. A confirmation window will appear, see below.
     

    Save Files Confirmation
