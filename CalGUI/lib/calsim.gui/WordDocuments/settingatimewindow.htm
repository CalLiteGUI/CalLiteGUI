Displaying and Editing Data


Setting a Time Window

You can specify the time window for data you want displayed in plots and tables and to perform math functions on.  For example, if you have a set of HEC-DSS records ranging from October 1, 1993, to December 30, 1993, you can specify a narrower time window such as 05OCT1993 to 25OCT1993 for plots, tables, and operations with math functions.

To set the Time Window:

  1. Select the Display → Time Window menu.  The Set Time Window dialog box will open (see below).
  2. You may select no time window, which will retrieve all the data for a data set, enter specific dates and times, use a time window relative to the current time, or access data by water year.
  3. To access all data in a data set, click the No Time Window button.
  4. To set a specific time window, click the Specific Time Window button.

§  If you wish, you can set the fields to the current time by pressing the Set Current Time button and then change what you want manually.

§  To specify dates and times:

      1. Enter the Start Date and End Date.  Although a variety of formats will be recognized for the date, the typical format is DDMMMYYYY.
      2. Enter the Start Time and End Time in 24-hour format (e.g., for 2:00 PM, type “1400”).

§  You may select the starting and ending dates from the calendar tool, which is accessible from a button on the right side of the date field.  When you click on this button, the selection calendar in a monthly format will opened (see the second figure below).  You may scroll through months or years by using the appropriate arrow key at the top of the box.

  1. To set a time window relative to the current time, click the Relative to Current Time button.
       
      1. Select the units you want to go back and forward from the current time.  You can choose hours, days, months, or years.
      2. Enter the Go Back and Go Forward quantity.
      3. If you do not set a quantity, No Time Window will be set.
            
  2. To access data by water year, select the Individual Water Year button.  You can change the default start date of the water year from October first by setting a new date in the field under the button.  This option will read all the data in a data set and then separate it into individual data sets by water year.  This is primarily useful for operations in the Math Functions dialog.  If you plot this data, you should choose Plot Individual Data Sets.
  3. To keep your setting the next time you run HEC-DSSVue, select the Retain Between Sessions checkbox.  The next time you execute HEC-DSSVue, your time window will be set.
  4. Click OK to apply the time window and close the dialog box.
