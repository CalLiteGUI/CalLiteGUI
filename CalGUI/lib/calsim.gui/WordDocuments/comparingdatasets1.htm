Comparing Data Sets

If you have similar time series data sets, you can compare them and find the differences.  The Compare functions will tell you how many values are different and highlight them in a table.  You can set options in the compare functions to allow a tolerance between values or compare only where there are valid values for the same times.

You can compare data sets from the Compare data sets menu option from the Edit menu (see below) or by selecting the pathnames for those data sets and using the Compare menu option from the Tools menu.

To Compare Data Sets:

  1. At least two pathnames must be selected for this menu option to be available.  Select the records you would like to compare from the Selected Pathnames List.  Select the Tools → Compare → Data Sets menu option. 

           Or

       From a table with the two data sets tabulated, select the Edit → Compare data sets menu option.

  1. If you are comparing from a table, a  message will appear displaying the number of values that differ in the selected data sets (see below) and the differences between the two data sets will be highlighted in red, as shown in the second figure below.

Comparing data sets can be customized to include tolerance levels, ignoring of missing data, and set to compare only values with matching time steps.   To use these options, you must use the Compare option from the Tools menu.

To compare Data sets using these options:

  1. At least two pathnames must be selected for this menu option to be available.  Select the records you would like to compare, displayed in the Selected Pathnames List.
  2. Select the Tools → Compare → Data Sets with Options menu option. 
  3. The Compare Options dialog will appear, see below. 


    Compare data set options
     
  4. To ignore all missing values during the compare, check Ignore Missing Values.  To only compare values at match time steps, check Coincident Times Only.
  5. In the Allowable Difference area of the Compare Options dialog, you can change the allowable tolerance for the comparison.  By default the comparison uses no tolerance; the Exact option is selected.  Instead of using the exact value, you can choose to allow a percent by selecting Percent Difference and entering a percent value from 0-100 in the field.  To set a numerical difference, select Amount Difference and enter a value in the field.
  6. When you are finished selecting Compare Options, click OK.
  7. The Compare Data Sets message will appear displaying the number of values that differ in the selected data sets, see below.


    Compare Data sets window
     
  8. To view a table of only the values that differ, click Show Only Differences.  To view a complete table of the data sets, click Show Full Table.
