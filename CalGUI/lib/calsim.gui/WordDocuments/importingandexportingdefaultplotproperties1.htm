Customizing Plots

Importing and Exporting Default Plot Properties

Plot property changes made in a plot’s Default Plot Properties editor apply to all plots used within the HEC-DSSVue application.  However, if you would like to use the same defaults for another HEC applications, you can import or export the properties.  The properties must be moved from/to a compatible HEC “Next Generation” java based software application.

 

Examples of compatible software are: HEC-DSSVue, HEC-FIA, HEC-ResSim, HEC-RAS, HEC-HMS, CWMS, etc.
