Importing Data from Microsoft Excel or “.csv” files using Data Entry

You can import time-series data from Excel, a “.csv” file, or a blank separated file using the Data Entry dialog.  At this time, you can only import time-series data (both regular interval and irregular interval data).  Check the HEC-DSSVue plug-ins web page for updates.

  1. Open your DSS file and select Excel from the Import popup menu from Data Entry on the menu bar, as shown below.
  2. If your data is in a “.csv” or blank separated file, select Time Series Data instead of Excel. An example of the browsing window is shown in the second figure below.


    Importing Excel or .csv Files
     
  3. Select your file to import.  Data should be in columns, separated by comma or spaces.  Ideally, a row should contain a location or parameter, and a column should contain date/time.


    Selection of an Excel or .csv File to import into HEC-DSSVue
      
  4. The file will be read into an editable table as shown below.


    Editable Import Table of an Excel or .csv File
              
  5. For regular-interval time series data, select the column cells that you wish to import.  For irregular-interval data, you need to select date/time column cells also.


    Selecting and Importing Data
       
  6. Press the Import button and your selected cells will be entered into the time series data entry dialog, as shown above.


    Time Series Data Entry window
      
  7. Enter the pathname parts, units and type, and then press Save.  An example of the Time Series Data Entry window that appears after data have been imported is shown above.

    Note:  Excel uses beginning of period dates, whereas HEC-DSS uses end of period.  Thus, the average data for a day will show at 0000 hours in Excel and 2400 hours in HEC-DSS.  This can cause a time shift by a day if the data is not imported or exported carefully; Check your data after the operation.  You can shift data to the correct times by using the Time Functions tab in the Math Functions screen.
