Exporting SHEF

You can export SHEF data by:

  1. Select the pathnames for the data sets that you want to export, then from the Data Entry menu select Export and then SHEF.
  2. A file selection browser will be displayed, as shown below.  Navigate to the directory you wish to export the data to, type the name of file and Press Save.  All data will be saved in one file.
