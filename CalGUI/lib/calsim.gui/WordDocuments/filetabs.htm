File Tabs

 

Multiple DSS files can be opened and viewed in HEC-DSSVue at the same time (see above).  Each time you open a new dss file, a tab is created with the name of the dss file shown on it.  The pathnames displayed in the Pathname list are from the selected tab’s DSS file.  If more than one file is opened, selected pathnames will include the name of the DSS file they are from.  For example:

 

BaldEagleHist.dss://BALDE/FLOW/01DEC1993/1HOUR/BB/
