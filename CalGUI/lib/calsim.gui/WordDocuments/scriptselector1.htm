Scripting > Executing Scripts


Script Selector

The Script Selector (see below) displays buttons for all the available scripts which have the Display Script on Toolbar box checked in the Script Editor (see the second figure below).   Buttons are displayed in alphabetical order.

 

Script Selector

 

Show in Scripts Menu Checkbox in Script Editor

 

To access the Script Selector, select the Script Selector command from the Utilities menu of HEC-DSSVue.  Once the Script Selector is open, it will remain open until you close it.

 

When you press a button, the Jython script engine will execute the associated script.
