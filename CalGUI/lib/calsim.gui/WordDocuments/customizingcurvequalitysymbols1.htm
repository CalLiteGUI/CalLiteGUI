Customizing Curve Quality Symbols

The curve Quality Symbols worksheet, available from the Curves tabs of the Edit Curve Properties editor, the Plot Properties editor, and the Default Line Styles options editor is shown in the figure below.  This worksheet allows you to customize curves for plots that have quality set for their data.

 

Curve Quality Symbols Editing Interface

 

To customize quality symbols:

  1. Check the Draw Quality Symbols checkbox.
  2. Select a symbol Style, Line Color, Fill Color, and Size for each quality of data; Valid, Questionable, Rejected, and Missing.
