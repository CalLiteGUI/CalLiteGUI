Exporting Data to Microsoft Excel

To export data to Excel, simply select the data sets you want to export and then press the Excel Icon   on the tool bar, or from the Display menu, select Tabulate in Excel.  An example of data tabulated in Excel is shown below. You can then save the Excel file to the name that you desire.  Currently you cannot format the data in a different manner.

If the Excel icon is not displayed, then you need to download the Excel.jar file from the web page listed above and save it in the HEC-DSSVue\Plugins directory.  This capability assumes that you have Microsoft Excel loaded on your computer or another program that is associated with “.xls” files.
