Importing SHEF Data

Once the SHEF tables are setup, you can import SHEF data by either dragging SHEF files with an extension of “.shef” or “.shf” from Windows Explorer and dropping them into the main screen of HEC-DSSVue with an opened HEC-DSS file or by selecting SHEF from the Import menu under the Data Entry menu.

To import SHEF data files into HEC-DSS:

  1. From the Data Entry menu select Import and then SHEF.
  2. A file selection browser will be displayed, as shown below.  Navigate to the directory and select the files you wish to import.  Select the files to import and press Open.  You can import multiple files at one time.  The file extensions do not matter for this option (the files do not need to end with “.shef”).


    Importing SHEF Data
      
  3. A message dialog will show how many values were stored from the SHEF data sets.  You can view the SHEF process log by pressing the View Log button, as shown below.


    Import Status

Or, alternatively you can import SHEF files if they have the extension “.shef” or “.shf” by:

  1. From Windows Explorer, select the “.shf” or “.shef” files that you wish to import.


    Selection of .shf or .shef Files to Drag and Drop into DSSVue
      
  2. “Drag” the files onto the main HEC-DSSVue screen (with your HEC-DSS file that you want to import to opened.)
  3. The data is imported and a message dialog will show how many values were stored from the SHEF data sets, shown below.  You can view the SHEF process log by pressing the View Log button.
