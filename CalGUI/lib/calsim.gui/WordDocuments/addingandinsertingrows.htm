Adding and Inserting Rows

 

Depending on what type of data is being edited in a table, rows can be inserted, appended, or added to the beginning or end of the table.

 

Appending rows is automatic when Allow Editing is set.  When the Edit → Allow Editing menu option is checked, two blank rows automatically appear at the end of the table.  When ever you enter data in a row at the bottom of the table, an additional blank row will be appended.  When you save the data, any blank rows at the end of the table are removed automatically. 

 

Extend Data Set window for regular interval data

 

For regular interval time series data, rows can only be inserted at the beginning or end of the data set.  For irregular interval time series data and paired data, rows can be inserted between any existing table row.  To insert a row into the table:

 

  1. Using your mouse pointer, click on the row you would like to insert rows next to and then select the Edit → Insert Rows menu option.  
  2. The Extend Data Set dialog will display, see figure above. This dialog may be different from what’s pictured in the figure above depending on what type of data is displayed in the table.  
  3. For irregular interval time series data, see the figure shown below, you have the option to Insert between the selected dates, Insert before the first date, and Append after the last date. Regular interval data can only have data inserted before and after the existing time steps.  Each choice requires you to enter a Number Rows value.  The Number Rows value represents the number of rows that will be added to the table at the given area.  You can enter a value directly in the Number Rows field or use the up and down arrows.


    Extend Data set window for irregular interval data

  4. Irregular interval time series data requires an Interval to be selected.  By default the 1HOUR interval is set.  Select an Interval using the corresponding dropdown list.  If you would like the data and time filled for each inserted row, be sure the Set the data and time for each row checkbox is checked.
  5. Paired data tables row inserts require only entering the number of rows you would like to insert, see the figure below.  Paired data tables also allow you to add additional columns through the Edit → Insert Column menu option.


    Insert Rows window for paired data

  6. Once you are done entering information for the insert, click OK.  
  7. The entered number of blank rows will be inserted at that row, moving the original row that you selected down.
