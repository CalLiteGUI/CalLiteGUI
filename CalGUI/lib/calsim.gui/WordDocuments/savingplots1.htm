Saving Plots

You can save a plot as an image as a Windows Metafile (*.wmf), Postscript (*.ps), JPEG (*.jpg, *.jpeg), or Portable Network Graphics (*.png).

 

To save a plot as a graphic file, from the File menu, click Save As.  From the Save dialog box (see below), select the location where you want to save the plot, enter a filename in the File name box, and select the file type you want from the Files of type list, then click Save.
