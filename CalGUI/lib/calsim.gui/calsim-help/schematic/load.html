Load a schematic

Click on File | Load Schematic, a file dialog box will pop up, open a schematic file, now a schematic should be displayed in the center of the gui. Right click on any of the objects, a popup menu contains all the possible data types should be displayed, selecting any of them will cause a corresponding data being displayed.

 

Note: You should always set the project to ready first. Please see "Prepare a project for study".
