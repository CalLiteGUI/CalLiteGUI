Return Flow Arc Table

Return flows in CALSIM are modeled as a fraction of the actual delivery to a particular node. The fraction and associated delivery arc are entered in this table to describe the return flow relationship.  Similar to the delivery arc table, the return flows can be specified as a constant fraction, or assigned values from “LOOKUP” tables or from TIMESERIES dss files.  The word “WRESL” in the return fraction column indicates that the return flow is determined in separate WRESL statements.
