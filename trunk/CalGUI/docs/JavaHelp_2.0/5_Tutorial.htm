5. Tutorial

5.1. Installation

The CalLite model is currently delivered in a single installer file. Installation consists of extracting all files in the archive to a file system location on a Windows XP, Vista, or Windows 7 computer. During installation, a CalLite3.00.exe icon is generated on the desktop. The software is started by double-clicking the icon. Alternatively, user can start the CalLite GUI from start>program>CalLite3.00>CalLite3.00. User can also start the CalLite GUI by double-clicking on the file “CalLiteGUI.exe” in the root directory of the extracted version (Figure 5-1).
