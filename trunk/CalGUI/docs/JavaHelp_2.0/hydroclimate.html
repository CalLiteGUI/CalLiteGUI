3.2. Hydroclimate Dashboard

The Hydroclimate dashboard allows the user to set the following:

-Climate Projection Period. The current version of CalLite has the capability to run simulations for the Current Level of Development (LOD) (2005), or Future (2020) LOD under the Historical Hydrology category, or future hydrologic periods of Early Long-Term 2025 or Late Long Term 2060 under the Climate Change category with different assumptions in regard to the land use and quantity of water demands.

-Sea Level Rise Options.  The different options are the current sea level, a sea level rise (SLR) of 15 cm, and a SLR of 45 cm.  The ANN X2 notation at the end of these labels signifies that the method for estimating the X2 (salinity of 2 parts per thousand) position is using the Artificial Neural Network (ANN) method.  See Appendix D of the CalLite Reference Manual for more information on sea level rise and X2 position simulation in CalLite.

-Climate Change Scenarios.  This version of the model has the capability to simulate operations using historical hydrology and different climate change futures.   CalLite allows modeling of the five climate changes scenarios (Q1-Q5) used in the BDCP analysis. These scenarios were determined by mapping 112 future climate projections. and obtained from 15 different Global Climate Models developed by various national climate centers.  Check Appendix E of the CalLite Reference Manual for more information.

-SV/Init Files.  As default, GUI selects the input hydrology file, also known as State Variable (SV) DSS file, based on Run Basis, Level of Development and Climate Projection Period. The CalLite GUI also selects the default initial condition DSS file. The user can select these input files in the current version.
