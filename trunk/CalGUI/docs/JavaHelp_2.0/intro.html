1. Introduction

 

The CalLite 3.00 User's Guide describes all of the screens of the CalLite GUI, to facilitate easy use of the model.  For information on aspects of the model other than the graphical user interface (GUI), see the CalLite 3.00 Reference Manual, which is distributed as a separate pdf. 

 

1.1.  The CalLite 3.00 Model  

 

CalLite is an interactive screening model for Central Valley water management, which simulates the hydrology of the Central Valley, reservoir operations, State Water Project (SWP) and Central Valley Project (CVP) operations and delivery allocation decisions, Delta salinity responses to river flow and export changes, and habitat-ecosystem indices.  CalLite contains much of the same functionality as the more detailed CalSim II model, but is simplified in its spatial and hydrologic detail.  The power of a screening model such as CalLite is the ability to rapidly simulate system operations and to incorporate changes with relative ease. The current version of CalLite simulates system operations on a monthly basis for the full 82-year period (Water year 1922-2003) of record in less than 5 minutes, and can be run with a variety of different options in terms of hydroclimate, demands, facilities and infrastructure, regulations, and system operations.  But despite the simplifications included in CalLite, the model results correlate very well to those of a comparable CalSim II study.

 

CalLite 3.00 is the newest version of the CalLite model, which is implemented using (1) a simulation engine developed using the Water Resource Integrated Modeling System 2 (WRIMS 2), and (2) a customized GUI that replicates all of the functionality contained in previous versions of CalLite.  WRIMS is the modeling platform used for the CalSim II model, and implementing CalLite in WRIMS will improve the consistency of CalLite and CALSIM results and make it easier to maintain that consistency as both models evolve.  The goal of Version 3.0 of CalLite is to provide the same capabilities and user-friendly interface as previous versions of CalLite, while maintaining a modeling structure that is as close as possible to the more detailed CALSIM model.  

 

Further information on the CalLite 3.00 model can be found in this User's Guide, in the CalLite 3.00 Reference Manual, and at the CalLite website.

 

A CalLite tutorial is available at the end of this User's Guide.

 

1.2.  Installation

The CalLite model is currently delivered in a single installer file. Installation consists of extracting all files in the archive to a file system location on a Windows XP, Vista, or Windows 7 computer. During installation, a CalLite 3.00.exe icon is generated on the desktop. The software is started by double-clicking the icon. Alternatively, user can start the CalLite GUI from start>program>CalLite3.00>CalLite3.00.
