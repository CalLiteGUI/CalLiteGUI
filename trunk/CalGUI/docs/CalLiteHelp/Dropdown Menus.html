2. Main Menu Options

The CalLite GUI has four main menu options:

File: Has options to create, load, save, run, or delete an existing scenario.

Scenario Settings:  Has 6 tabs for setting parameters for the current scenario, described in Section 3 of this User's Guide.

Results: Has 4 tabs for viewing results, described in Section 4 of this User's Guide.

Help: Provides access to this help system and the current CalLite version.
