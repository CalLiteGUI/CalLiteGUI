3. Demand screen

 

 

The demands screen allows the user to make selections relating to SWP and CVP SOD demands. 

For SWP, the user can select variable Table A demand levels, full Table A demand levels, and user-defined demands.  When selecting user-defined demands, the specific demand values can be specified for MWD, Other M&I, and AG demands.  The values entered here cannot exceed the full Table A amount.  The user can also enter proportion of Article 21 interruptible demands to deliver.  When selecting user-defined demands, values for losses are held constant, and Article 56 (carryover) demands are set = 0.

For CVP, the user can selected full contract demands or user-defined.  When selecting user-defined demands, the specific demand values can be specified for AG, MI, and Refuge demands.  The values entered here cannot exceed the full contract amount.  When selecting user-defined demands, values for losses are held constant, and water rights and exchange contractor deliveries cannot be altered.
