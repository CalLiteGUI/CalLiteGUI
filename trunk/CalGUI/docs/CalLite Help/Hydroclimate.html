3.2. Hydroclimate Dashboard

The Hydroclimate dashboard allows the user to set the following:

-Level of Development (LOD) to 2005 or 2030.  Different LODs have different assumptions with regard to land use and the quantity of water demands.

-Sea Level Rise Options.  The different options are the current sea level, a rise of 15 cm, and a rise of 45 cm.  The ANN X2 notation at the end of these labels signifies that the method for estimating the X2 (salinity of 2 parts per thousand) position is using the Artificial Neural Network method.  See Appendix D of the User's Reference Guide for more information on sea level rise and X2 position simulation in CalLite.

-Climate Projection Period.  The current version of CalLite only runs simulations with historical hydrology (1922-2003).  Future hydrologic periods will be added in future versions of the model.

-Climate Change Scenarios.  This option is not active in the current version of CalLite, but will be added in future versions.
