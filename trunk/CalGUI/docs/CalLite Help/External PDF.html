4.4. External PDF tab

This tab allows the user to produce a standardized pdf report that compares the results of two CalLite scenarios.  The report shows storage in major reservoirs, flows at key river segments, CVP and SWP deliveries, and other scenario outputs.  In order to generate the report, select a report template file, two DSS output files, a report output file name, and hit the Generate Report button.  Optional information such as notes and modeler name can also be added to the report.
