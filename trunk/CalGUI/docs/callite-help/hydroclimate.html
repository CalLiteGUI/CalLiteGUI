3.2. Hydroclimate Dashboard

The Hydroclimate dashboard allows the user to set the following:

-Level of Development (LOD). CalLite offers current (2005) and Future (2020) LODs which have different assumptions with regard to land use and the quantity of water demands.

-Sea Level Rise Options.  The different options are the current sea level, a rise of 15 cm, and a rise of 45 cm.  The ANN X2 notation at the end of these labels signifies that the method for estimating the X2 (salinity of 2 parts per thousand) position is using the Artificial Neural Network method.  See Appendix D of the CalLite Reference Manual for more information on sea level rise and X2 position simulation in CalLite.

-Climate Projection Period.  The current version of CalLite only runs simulations with historical hydrology (1922-2003).  Future hydrologic periods will be added in future versions of the model.

-Climate Change Scenarios.  This option is not active in the current version of CalLite, but will be added in future versions.

-SV/Init Files.  As default, GUI selects the input hydrology file, also known as State Variable(SV) DSS file, based on Run Basis, Level of Development and Climate Projection Period. The CalLite GUI also selects the default initial condition DSS file. In future, user will be able to select these input files.
