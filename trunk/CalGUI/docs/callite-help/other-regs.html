3.5.3. Other Regulations Tab

This tab allows the user to activate three additional regulations and define values for them in the table on the right side of the screen.  The month numbers in the user-defined tables refer to USGS water year months, so Oct = 1, Nov =2 etc.        

The regulations available here are:

-QWEST (San Joaquin River near Jersey Point). This is a minimum flow standard on the San Joaquin River near Jersey Point.  The user can define values for this standard by month and water year type.

-Old and Middle River.  This limits CVP and SWP exports so that flow in the Old and Middle River (OMR) is no more negative than a specified standard during Dec-June.  Standards can be specified by month and water year type.  When this standard is activated here, the OMR standard on the Biological Opinion RPAs tab is automatically de-activated.

-San Joaquin River Inflow to Export ratio with offset.  This limits combined CVP and SWP exports to a cap computed based on San Joaquin River flow at Vernalis.  The cap is computed as = offset + multiplier * Vernalis flow.  The offset and multiplier values can be specified by month and water year type. When this standard is activated here, the SJR inflow/export ratio standard on the Biological Opinion RPAs tab is automatically de-activated.

-Fraction of Unimpaired Flow for Delta Criteria.  In July 2010 the State Water Resources Control Board issued a report on Delta Flow Criteria.  Here the user can activate those criteria in the model (for Delta outflow, the Sacramento River, and the San Joaquin River) and also specify different criteria if desired.  Note that unlike all of the other regulations on this dashboard, when this option is activated, CalLite does not force flows at these locations to meet these criteria.  It only compares flows to the criteria and computes how much additional water would be needed to meet the criteria, if it is not met.

WARNING: Any user defined values will be used in place of all D1641 and Biological Opinions regulations. This applies to Old Middle River and San Joaquin River Inflow/Export ratio user defined options.
