4.1. Quick Results tab

The Quick Results tab is where graphs and tables can be generated for numerous CalLite outputs, for one scenario or comparing different scenarios.  The different sections on this tab are as follows:

-Scenarios.  Here the user can add or delete scenarios from a list of scenarios whose results will be displayed.  The radio button next to each scenario name can be used to select which scenario to delete, and also sets the base scenario for the difference time series plot.  Here one can choose the type of monthly time series plot to display (either a comparison of scenarios or the difference between scenarios)

-Display.  Has a series of display options, which are:

-Starting and ending months to display.  This setting only affects the time series plots.

-Whether to show data in cfs or taf.

-Whether to show a time series or exceedance plot.  Exceedance plots can be shown for individual months or all monthly values.

-Whether to show a table of monthly values.

-Whether to show a summary table.  The summary table can show five different statistics summarized by four different water year types and also for the three driest periods in 1922-2003.

-Output selection.  The Storage and Flows tab allows the user to choose from a variety of standard output variables.  Double-clicking on the variable name will display graphs and tables for that variable.  The Water Management Action tab is currently not active.

-Report List.  This section can be used to create reports which include different selections of multiple graphs and tables.  Clicking the Add to List button will add a report to the list that includes all the variables whose checkboxes are active with all the options selected in the Scenario and Display areas.  Reports on the list can then be displayed (singly or all at once), printed, and saved for later use.

When graphs are displayed it is possible to zoom in by clicking and dragging the cursor to the lower right to create a box.  If you are zoomed in, clicking and dragging the cursor to the upper left will zoom you all way back out. Right clicking in the graph area will bring up a menu that allows you to edit properties of the graph; copy, save, and print the graph, and zoom in and out.
