Listing DTS:
   The List sub menu lists all the DTS names in the current project, and it dynamically changes when you add or delete DTS to this project. Click on DTS menu, highlight List sub menu, you will see all the DTS are listed, click on any DTS, a DTS table which contains the contents of this DTS will be displayed.

Note: The List, Retrieve, and Delete sub menus always have the same DTS name list.

Also see Retrieving DTS, Deleting DTS.
