Working with General retrieve

General menu will allow you to freely access any data in a DSS file. Select General tab in the central panel of the Output tab. In the upper portion of the panel, choose a variable (file) type in the combo box, type in the specific pathname parts in the corresponding text fields. Now click on Filter button, you will see all the path names math the criterion are displayed in the table. Then select as many pathnames as you want, then click on Retrieve button to display the data in the format you selected in the View menu.

Example: Click on General tab, type "flow" in the text field of "C-part", then click on Filter button, you should see all the pathnames with "FLOW" as C-part are displayed in the table. Lets select the first and the third pathnames, click on Retrieve button, a multiple data sets table will be displayed.

Note: You should always set the project to ready first. Please see "Prepare a project for study".
