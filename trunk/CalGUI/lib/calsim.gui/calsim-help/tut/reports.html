Working with Reports

The Reports menu contains items for each customized report. Many reports will be added in the future, for now we only have Water Supply Impact and Delta Balance. Select Reports toggle button first, then select a menu item and click on it, a customized report will be created and displayed.

 

Example Click on Reports toggle button, click Reports | Water Supply Impact, a report will be displayed.

Note: You should always set the project to ready first. Please see "Prepare a project for study".
