Additional Viewing Options for Plots

 

File Menu

 

The File menu of plots (see above) contains several commands that allow you to view plot data in tabular form, save, copy, paste, and print plots.  Copy can be used to copy the plot into other applications such as Microsoft Word, Excel, and Visio.
