SHEFPARM

The SHEFPARM file contains SHEF PE (Physical Element) codes and the metric – English conversion factor.  Generally, the supplied file should be sufficient unless newer PE codes are being used.   In this table, the first column contains the PE code and the second column contains the Metric to English conversion factor, as shown below.  If there is no factor, set this cell to 1.0.
