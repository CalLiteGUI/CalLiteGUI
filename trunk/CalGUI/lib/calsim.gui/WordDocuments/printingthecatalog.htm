Printing the Catalog
 


Print Preview window

You can Print the table catalog or show what the catalog will look like when it is printed, from the Print Catalog Preview command or from the Print Catalog command from the File menu.  Each command opens the Print Properties dialog box (see below), which offers options on three tabs.

A catalog for the DSS file consists of a list of the pathnames contained in the file, similar to what is displayed in the DSS Pathname list area.  The View menu settings for the DSS Pathname list will determine how the catalog will display the pathnames.  For example: as a sorted list or by pathname parts.  The catalog for the currently selected DSS file can be printed or viewed in a separate window on your desktop. 

To preview the catalog in a separate window, in the File menu select Print Catalog Preview….  To print the catalog, select File -> Print Catalog.  Both commands will open a Print Properties dialog (see below), which offers print options on three different tabs.  On the Print Properties dialog, click the Print button.  If you’re viewing the Print Properties dialog for a Print Preview, click the Preview button to view the catalog in the Print Preview dialog.  From the Print Preview dialog (see above), you can print the catalog by clicking the Print button located in the upper left-hand corner of the dialog.


Print Properties Dialog Box

The Page tab allows you to specify the page Orientation, Scaling, and Selection; you can also choose to print the table as ASCII, Repeat Headers on every page, and print the Gridlines.

On the Header/Footer tab, you can type in the header and footer you want to appear on your printed pages.

The Table Title tab offers a default title for the table based on the data source.  You may edit this title.

On the Print Properties dialog box, the Print button performs two functions, depending on whether you arrived at the dialog box via the Print command or the Print Preview command.

You can also print traditional versions of the catalog from the Advanced menu.
