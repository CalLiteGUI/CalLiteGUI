Scripting

You can use Jython scripts that you or others write to perform math functions, create custom plots or tables, and automate repetitive tasks.  To edit, test, and select scripts, use the Script Browser.  To run scripts that have already been setup, use the Script Selector window.  To access the Script Browser (see below), from the Tools menu, click Script Editor.  To access the Script Selector window, from the Tools menu, click Script Selector.  For more information, refer to the chapter on Scripting.
