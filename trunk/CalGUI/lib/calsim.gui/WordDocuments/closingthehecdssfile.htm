Closing the HEC-DSS File

To close the currently selected HEC-DSS database file, from the File menu, click Close.  This will allow some older MS-DOS based programs to access the file or for you to rename or delete the file. 

 

Note -  HEC-DSS is a multi-user database system.  Some older MS-DOS programs exclusively locked HEC-DSS files.

HEC-DSS will automatically close un-accessed file files after about a minute.  They will be automatically re-opened on the next access.
