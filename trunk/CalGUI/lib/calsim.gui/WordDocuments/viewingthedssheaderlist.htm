Viewing the DSS Header List

The DSS Header List displays all the database variables and their values used for the current HEC-DSS file. 

To view the Header List:

  1. After opening an HEC-DSS File, select the Advanced → Status → DSS File Header menu option.
  2. A dialog will open listing all the headers set for the current HEC-DSS file, see below.
