Displaying and Editing Data


Tabulate and Editing using MS Excel (Optional)

If the Excel plug-in is loaded, you can tabulate (or export) data in Microsoft Excel and edit time series data in Excel.  Simply select the data sets you wish to tabulate as you normally would and then either press the Excel icon in the toolbar, or select Tabulate in Excel from the Display menu (assuming you have Microsoft Excel installed). The figure below is an example of how the data is displayed in Excel. 

To edit data in Excel, select your data sets and select Edit in Excel from the Edit menu.  This will load the selected data into an Excel spreadsheet and then read it back to DSS once you have saved it.  In this mode, you may not add or delete rows or columns, you can only change the data.  After you are finished, save the Excel spread sheet and then press OK or Cancel from the dialog from HEC-DSSVue.

Note – Take caution with dates and time in Excel.  Excel shows times as “Beginning of Period”, while the HEC-DSS convention shows times as “End of Period”.  A data value for midnight will show up as 2400 hours at the end of the day in HEC-DSS, while Excel will display that as 0000 hours for the next day.  For daily data, the hours might not be shown in Excel and the data can be misinterpreted to be for the following day.
