List of HEC-DSS Pathnames

Once you have opened an HEC-DSS file, its pathnames appear in a pane beneath the Search By filters.  The pathnames can be displayed as a pathname sorted list (Pathname List), by DSS Parts (Pathname Parts), in the Condensed Catalog format, with No Pathnames* displayed, or as an Unsorted List*.  These options are set from the View menu, as described in section "Choosing a Display Mode for HEC-DSS Pathnames".  An example of Pathname by Parts is shown below.

 

*Note - The No Pathnames option is typically used to access very large databases using only scripted functions.  The Unsorted List option can be used to view pathnames in very large databases without the time required to sort the pathnames.
