HecTime Class

HecTime objects are used to manipulate dates and times and to convert dates and times among different formats.  To use HecTime objects, the HecTime class must be imported from the hec.heclib.util module (e.g. “from hec.heclib.util import *” or “from hec.heclib.util import HecTime”).  After importing the class, a new HecTime object can be created by calling HecTime() (e.g. “myTime = HecTime()”).  HecTime methods are described below.
