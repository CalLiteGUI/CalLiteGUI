Editing Data in Microsoft Excel

You can edit data in Microsoft Excel by selecting the data sets you want to edit and then from the Edit menu, select Edit in Excel.  The data will be tabulated.  After you have made you changes, save the Excel file and then press OK in the edit dialog that is displayed.  If you do not want to save your changes back to HEC-DSS, press Cancel.

Note:  You can only change data in Excel when you are editing it – you cannot add or delete data.  The data must remain in the same format with the same number of rows and columns to be stored back to HEC-DSS.
