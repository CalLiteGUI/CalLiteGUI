Searching for Values

You can search for specific values in your data set by selecting the Find option from the Edit menu or by pressing the F3 key (see below).  Selecting this option will display the search dialog, as shown in the second figure below.

 

Tabulate Edit menu

 

Find window

 

When you press Find Next, the next value that matches will be highlighted.  You can search up from where your cursor is, or search down.  You have the option of searching for the exact number or part of that number.  You can also search for a date or time.
