HecMath Class

The objects returned from the HecDss..read(…) methods and supplied to the HecDss.write(…) method are HecMath objects.  There are currently three types of HecMath classes: TimeSeriesMath class, PairedDataMath class, and StreamRatingMath class which represent time-series data, paired data, and stream rating data, respectively. 

 

An HecMath object can be created for writing to new DSS data by first creating a new DataContainer as discussed in the sections "TimeSeriesContainer Class" and "PairedDataContainer Class", and then calling the HecMath.createInstance() function with the DataContainer object as the only parameter (e.g. myTimeSeriesMath = HecMath.createInstance(myTimeSeriesContainer)).  The methods for HecMath Objects, are listed in below.
