Manual Text Entry

HEC-DSS has a text convention format.  You can enter data by manual text data entry and type or paste text into the text editor, or you can import a text file.  You cannot import text files into the HEC-DSS text format by dragging and dropping a .txt file; this will save a .txt file in the generic file format.    

To enter text data manually:

  1. From the Data Entry menu, choose Manual Text.  The Manual Text Data Entry dialog box dialog box (see below) will open.


    HEC-DSSVue Manual Text Data Entry Dialog Box
       
  2. Type or paste the text data into the text editor box.  You can change the font, but the font will not be stored with the text.
  3. You can select the File menu and insert a text file into the editor.  You can also save your text as a .txt file, or save into the HEC-DSS file, or Print your text (see below).  You can also change the Font, but that will not be retained.  (Note:  If you want to save text data directly from a .txt file, use the Import Text from File menu under the Data Entry menu.)


    Print menu
      
  4. After the text has been entered, press the Save As button or the Save or Save As menu item from the File menu.
  5. Enter the Pathname Parts into the A, B, C, D, E, and F boxes.  The complete pathname will automatically appear in the Pathname box.  You can also enter the pathname into the Pathname box; the parts will appear in the Pathname Parts boxes, see below. 


    Save Manual Text Data
      
  6. To save the record, click OK.
