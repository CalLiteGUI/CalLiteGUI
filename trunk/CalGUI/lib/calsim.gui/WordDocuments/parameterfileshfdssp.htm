Parameter File – shfdssp

The parameter file links the HEC-DSS “C” part (the parameter), the data units, and type to the SHEF parameter (PE).  For each PE type in your SHEF data, a row should be present with the PE code in column 1, the HEC-DSS C part in column 2, the data units in column 3, the data type in column 4, and an optional factor to multiply the data by (for example, kilo-cfs might have a factor of 1000.0 and store the data in HEC-DSS as cfs.)  The sample parameter file that is distributed with HEC-DSSVue should be relatively complete.
