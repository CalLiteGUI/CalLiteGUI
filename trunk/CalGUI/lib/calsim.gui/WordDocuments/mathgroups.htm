Math Groups

You can open the Math Functions editor for a particular Group to perform math operations on the data sets within the Group.  For more detailed information on the Math Functions editor, see Math Functions.  To use the Math Functions editor, from the Groups → Math submenu, select the name of the group.  The Math Functions editor will open with the Group’s records populating the Selected Data Set lists.
