Displaying and Editing Data > Graphical Editing


Graphical Editor View Options

From the View menu, you can choose to plot the other data sets that you have selected on the same plot at the same time by selecting the Show Comparison Data item.  If the data types are the same, the data sets will plot in the same viewport; otherwise they will be plotted in different viewports.

You cannot edit the comparison data sets.  To edit comparison data sets, you must select a data set from the Selected Data Set box dropdown list.

You can also display a standard table of the selected data from the View → Tabulate menu option, or a standard plot from the View → Show Plot menu.  You can set the precision of decimal places for your data by selecting Decimal Places and selecting the number of decimal places you wish to display form the list in the submenu.
