Comparing Data Sets

If you tabulate two or more similar time series data sets, you can compare values from both sets from the table.  Select Compare Data Sets (see below) from the Edit menu.

A message will appear displaying the number of values that differ in the selected data sets (see the second figure) and the differences between the two data sets will be highlighted in red, as shown in the third figure below.
