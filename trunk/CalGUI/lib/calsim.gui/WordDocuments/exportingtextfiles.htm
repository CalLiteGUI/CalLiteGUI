Data Entry, Import and Export > Import/Export Files


Exporting Text Files

  1. You can export text files by selecting the pathname for the text data set that you want to export, then from the Data Entry menu select Export and then Export to File(s). You can only export one text data set at a time for this convention.  (For image and generic files you can export several at one time.)  This menu option will only appear for text, image and generic files.
  2. A file selection browser will be displayed, see below.  Navigate to the directory you wish to export to and type in the name of the file that you want the text to be written to.   Press Save.
