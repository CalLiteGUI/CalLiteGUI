Creating and Saving Groups

To create a Group:

  1. Select all the records you would like in the Group in the Selected Area.  If you set a time window, that will be saved along with the group
  2. From the Groups menu, select Save Selected.  The Input dialog will appear.  Type the group’s name in the Enter group name box (see below) and click OK.


    Group name box
       
  3. The Saved Group message (see below) will appear when the Group has been saved.  Click OK.


    Saved Group message box
       
  4. The Group will be saved and listed in the Groups submenus, see below.


    Group submenus
