Deleting Rows and Columns

You may delete rows from a table when working with irregular-interval time series data and paired data.  To delete a row from a table, select the row and then on the Edit menu, click Delete Rows.

If you are working with regular interval time series data, you will not be able to delete rows; instead, the Delete Rows function will change the data values to missing.  This is the same as selecting Clear from the right-click shortcut menu.

In a paired data table you can delete a column.  To delete a paired data column, select the column and then select the Edit → Delete Column menu option.  The Delete Column message will appear, requiring you to verify you would like to delete the entire column (see below).  Click Yes to finish deleting the column.
