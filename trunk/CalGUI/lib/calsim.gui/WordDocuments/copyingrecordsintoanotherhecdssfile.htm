Copying Records into another HEC-DSS File

To copy records into another HEC-DSS file:  

  1. Select the record or records to copy.
  2. From the Edit menu, select Copy To.  The Copy Records into HEC-DSS File dialog will open, see below.
  3. In the File name box, type in a new HEC-DSS filename or select an existing HEC-DSS File you want to copy the record into and click Open.  A confirmation message will appear stating that the record has been copied to the HEC-DSS file you selected. 
  4. If the record(s) already exist in that file, a message box will appear asking if you want them to be overwritten, see the second figure below  If the number of pathnames is four or less, their names will be displayed in the message box; otherwise they will be printed in the log screen.
