Script Editor

The Script Editor, shown below, allows you to add, delete, and modify scripts. 

 

Script Editor

 

You can access the Script Editor from HEC-DSSVue’s Utilities menu by clicking Script Editor.  Alternatively, you can get to the Script Browser from the shortcut menu of the Script Selector.  In the Script Selector, right click on a button to access the shortcut menu, and then select Edit.  The Script Browser will open with that script selected and ready for editing.

 

Components of the Script Editor include the Menu Bar, the Editor Panel, and the Tree Hierarchy.  The following sections describe these components.
